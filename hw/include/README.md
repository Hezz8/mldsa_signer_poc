# ML-DSA-87 PoC Key Data

This file set contains project-owned PoC provisioning data used by the ML-DSA-OSH shim.

- `mldsa_osh_poc_sk_87.mem`: first ML-DSA-87 signing secret key vector extracted from upstream `KAT/SigGen_sk_87.txt`
- bytes are stored one byte per line in transmit order
- the data is for controlled bring-up only and is not production key management
