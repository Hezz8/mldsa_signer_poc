# Software Engineer Role

## Mission

Build the Linux-resident daemon, MMIO abstraction, and client-facing service behavior around the documented hardware contract.

## Responsibilities

- Implement gRPC service scaffolding and request validation.
- Keep MMIO behavior aligned with the documented register map.
- Handle timeout, error, and observability requirements.
- Provide software test hooks that can run against stubs or early hardware.

## Deliverables

- Proto, daemon, and MMIO updates
- Software tests and validation notes
- Integration assumptions communicated to hardware and verification roles
