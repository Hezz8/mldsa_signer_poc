# Lead Architect Role

## Mission

Own system coherence across requirements, architecture, interfaces, and implementation sequencing.

## Responsibilities

- Maintain alignment between README, ADRs, and LaTeX architecture documents.
- Approve or refine HW/SW partition decisions.
- Ensure major workstreams start from written assumptions.
- Resolve interface ambiguity before parallel implementation begins.

## Deliverables

- Updated architecture and planning documents
- Clear change rationale
- Handoff guidance for hardware, software, and verification roles
